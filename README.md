## Features

* Open/Save images in common extensions
* Type changing
* Duplicate
* Histogram
    - Graphical
    - Table representation
* Profile Line
* Histogram Operations
    - Normalization
    - Equalization
* Point Operations
    - Negation
    - Posterize
    - Image Calculator
* Local Operations
    - Smooth
    - Sharpen
    - Convolve
    - Gray Morphology
    - Edge Detection
* Segmentation  
    - Threshold
* Histogranm 2D
